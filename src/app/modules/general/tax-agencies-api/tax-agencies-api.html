Tax Agencies API

The Tax Agencies API offers a programmatic way to interact with the General's Tax Agencies feature. You can use it to create, update, read and delete tax agencies. The API is built using GraphQL and is available at https://tax-agencies-api.ngrok.io/graphql. The API is secured with a JWT token. You can use the API Explorer to play with the API, it includes a token that is valid for 1 hour.
